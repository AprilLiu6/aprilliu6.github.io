# 理论篇

先记录下几个关键名词：

+ UI：User Interface
    介于 ***用户与硬件*** 产生互动时，而设计的让彼此之间产生 ***交互的相关软件*** 。最主要的就是 ***人机交互*** 和 ***图形用户界面***。

+ HCI：Human-Computer Interaction

+ GUI：Graphical User Interface

+ WUI：Web User Interface

+ IxD:Interaction Design

+ UX:User Experience

+ AD:Art Designer
